# Quickstart: Category Products Page

To test the implementation of the Category Products Page:

1. Run the application on an emulator or physical device.
2. Log in as a customer (or proceed to the main navigation screen if already authenticated).
3. Tap the "Categories" tab located in the bottom navigation bar.
4. Verify that the screen successfully loads and displays a list or grid of available categories.
5. Tap on any specific category (e.g., "Electronics").
6. Verify that the app navigates to the "Category Products" screen and displays the products associated with that category.
